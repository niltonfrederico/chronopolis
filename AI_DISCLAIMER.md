# AI Usage Disclaimer

## Project: Chronopolis

### Responsible AI Development

This project uses AI assistance (GitHub Copilot with Claude Sonnet 4) in a controlled and responsible manner. This document outlines the specific boundaries and ethical guidelines governing AI usage in this project.

### Permitted AI Usage

✅ **Documentation Generation**

- README files, API documentation, code comments
- Configuration explanations and setup guides
- Technical specifications and architectural decisions

✅ **Configuration Management**

- Build configurations (pyproject.toml, CI/CD files)
- Linting and type checking setups
- Development environment configurations

✅ **Code Completion & Small Functions**

- Auto-completion for standard patterns
- Small utility functions and helpers
- Type definitions and interfaces
- Test scaffolding and basic test cases

### Prohibited AI Usage

❌ **Large Code Generation ("Vibe Coding")**

- No AI-generated application logic or business rules
- No complete features or modules generated by AI
- No copy-paste of large AI-generated code blocks

❌ **User Data Processing**

- AI will never process, analyze, or handle user data
- No personal information sent to AI systems
- No database contents or user-generated content exposed to AI

❌ **Unrestricted External Integrations**

- Only GitHub MCP integration permitted for this project
- Monday.com MCP reserved exclusively for official work purposes
- No other external AI services or integrations

### Development Philosophy

**Human-Centric Development**: All critical decisions, architecture choices, and business logic remain under direct human control.

**Transparency**: All AI assistance is documented and reviewable through version control.

**Security First**: No sensitive data, credentials, or proprietary information is shared with AI systems.

**Quality Assurance**: All AI-assisted code undergoes the same review, testing, and quality standards as manually written code.

### Compliance & Oversight

- This disclaimer must be consulted before any AI assistance request
- Any request violating these guidelines must be declined with explanation
- Regular review and updates to maintain alignment with best practices
- All AI interactions logged through standard development workflows
